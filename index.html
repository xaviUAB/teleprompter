Benvingudes i benvinguts al vostre teleprompter. Premeu el botó de configuració per començar o carregar el vostre text.

Configuració del Teleprompter

Escriviu el text o pugeu un fitxer:

Dreceres de teclat:

• Espai: Pausa / Reprèn

• Dreta/Esquerra: +/- Velocitat

• Amunt/Avall: Avança / Retrocedeix

• Q / A: +/- Mida lletra

• Esc: Tanca menús

Com funciona?

  • Espai: Pausa / Reprèn el moviment.
  • Fletxes Dreta/Esquerra: Augmenta / Redueix la velocitat.
  • Fletxes Amunt/Avall: Avança / Retrocedeix el text manualment.
  • Q / A: Augmenta / Redueix la mida de la lletra.
  • Botó Gir/Inversió: Canvia l'orientació (Normal, Mirall, Girat).
  • Esc: Tanca els menús.
