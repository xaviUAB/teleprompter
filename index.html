Dreceres (Teclat, Ratolí i Tàctils):

• Espai / Clic roda / Toc (Tap): Pausa o reprèn

• Fletxes ↔ / Roda / Doble toc (Abaix Dreta/Esq): Ajustar velocitat

• Fletxes ↕ / Repàg/Avàg / Arrossegar: Desplaçar manualment

• N / M: Saltar al marcador anterior (N) o següent (M)

• Q / A o Doble toc (Dalt Dreta/Esq): +/- Mida de la lletra

• B: Obrir menú de marcadors

• Ctrl + R: Reiniciar i anar a l'inici

• Esc: Tancar qualsevol menú

Com funciona?

  • Autodesat: El text i els paràmetres es desen automàticament. No perdràs la feina si tanques la pestanya.
  • Marcadors de secció: Escriviu # Nom del marcador a l'inici d'una línia per crear-lo. Premeu N (anterior) o M (següent) per saltar-hi durant la lectura.
  • Pantalla completa: Utilitzeu el botó del menú inferior per llegir de forma immersiva.
  • Controls Tàctils: Un toc per pausar/reprendre. Doble toc inferior (dreta/esquerra) per ajustar la velocitat. Doble toc superior per la mida de lletra. Arrossegueu per moure el text amunt o avall.
  • Espai o Clic roda: Pausa / Reprèn el moviment.
  • Fletxes Dreta/Esquerra o Roda: Ajusta la velocitat del desplaçament automàtic.
  • Fletxes Amunt/Avall o Repàg/Avàg: Avança o retrocedeix manualment.
  • Q / A: Ajusta la mida del text proporcionalment.
  • Ctrl + R: Reinicia el guió a la primera línia.

Marcadors de Secció

Afegiu # Nom de la secció a l'inici d'una línia del guió per crear marcadors. Premeu N (anterior) i M (següent) per saltar entre ells de forma ràpida durant la lectura.
